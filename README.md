# rimi

My personal assistant. Rimi is intended to be a collection of utilities written in Ruby.

### Installation

Grant execute permission.

```sh
chmod u+x rimi
```

Add to `$PATH`.

```sh
ln -s ~/Documents/rimi/bin/rimi /usr/local/bin
```
